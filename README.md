# Comprar ou Investir

Simulador financeiro para comparar compra de imóvel, financiamento e investimento.

## Acesse

- Site: https://comprarouinvestir.app/

## O que este projeto faz

- Compara cenários de financiamento e investimento
- Permite configurar taxas e prazos
- Exibe resultados de forma clara e objetiva

## Como executar localmente

1. Abra o arquivo `index.html` no navegador, ou
2. Use um servidor estático (exemplo):

```
npx serve .
```

## Observações

Este projeto é uma SPA simples em JavaScript, com a lógica financeira centralizada em `calc.js` e a interface em `index.html` e `app.js`.
